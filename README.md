# Характеризация фемтосекундных лазерных импульсов ближнего инфракрасного диапазона методами машинного обучения
Для восстановления фазы и длительности фемтосекундных лазерных импульсов используются методы машинного обучения, основанные на нелинейных изменениях спектра при фазовой самомодуляции. Для обучения нейросетевых моделей используются смоделированные данные распространения импульсов в стеклянной пластине. Обученная на результатах численного моделирования нейросеть восстановила фазу и временную огибающую импульса из экспериментальных данных.
